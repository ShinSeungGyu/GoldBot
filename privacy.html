개인정보 처리방침 (Privacy Policy)

로스트아크 알림 봇은 사용자의 프라이버시를 중요하게 생각하며, 최소한의 정보만을 수집합니다.

1. 수집하는 정보의 항목

본 봇은 기능 수행을 위해 다음과 같은 정보를 디스코드 API를 통해 수집할 수 있습니다.

  • 디스코드 사용자 ID 및 닉네임
  • 서버(길드) ID 및 채널 ID
  • 명령어 입력 시 사용자가 직접 제공한 데이터

2. 정보의 수집 및 이용 목적

수집된 정보는 오직 봇의 기능 제공(예: 게임 데이터 조회, 자동 알림 송신 등) 및 서비스 운영을 위해서만 사용됩니다.

3. 정보의 보유 및 파기

수집된 데이터는 기능 수행에 필요한 기간 동안만 보유하며, 봇이 서버에서 삭제되거나 사용자가 요청할 경우 즉시 파기합니다.

4. 제3자 제공

본 봇은 원칙적으로 사용자의 정보를 외부에 제공하지 않습니다.

5. 문의처
