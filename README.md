# 🤖 Lost Ark Notification Discord Bot
로스트아크(Lost Ark) 유저들을 위한 디스코드 알림 봇입니다.
매일 정해진 시간에 오늘의 골드섬 정보를 자동으로 공지하고, 슬래시 명령어(/)를 통해 침공 보스 등의 게임 내 정보를 빠르게 확인할 수 있습니다.

# ✨ 주요 기능
📅 골드섬 자동 알림: 매일 오전 10시 30분, 서버의 #알림 채널로 오늘의 골드섬 정보를 자동으로 전송합니다.

⚔️ 경매장 최저가 조회: 매 정각마다 딜러/서폿의 종결 악세의 시세를 알려줍니다. 명령어를 통해 시세 변화를 그래프로 받아 볼 수 있습니다.


# 🛠 기술 스택
Language: Python 3.8+
Library: discord.py (v2.0+)
Framework: discord.ext.tasks, discord.ext.commands

# 📋 주요 명령어
| 명령어 | 설명 | 
| !골드섬 | 당일 존재하는 골드섬을 확인 할 수 있습니다. | 
| !경매장 | 봇 작동 시점부터 지금까지 저장된 악세의 값 변화 그래프를 받아볼 수 있습니다. |
